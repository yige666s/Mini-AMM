## 什么是 Mini-AMM

Mini-AMM 是一个简单、安全、自动化的数字资产交易平台。您可以：

- 💱 **交换代币**：将一种代币换成另一种代币
- 💰 **提供流动性**：存入资产赚取交易手续费
- 🤖 **自动复投**：让智能机器人自动帮您将收益再投资

### 核心优势

- ✅ **简单易用**：无需复杂操作，几步即可完成交易
- ✅ **低手续费**：仅收取 0.3% 的交易费用
- ✅ **自动增值**：收益自动复投，资产持续增长
- ✅ **透明安全**：所有交易记录公开可查

## 快速开始

### 第一步：连接钱包

1. 访问 Mini-AMM 平台
2. 点击右上角的 **"Connect Wallet"** 按钮
3. 选择您的钱包（如 MetaMask）
4. 在钱包中确认连接请求

### 第二步：获取测试代币

初次使用时，您的账户可能没有代币。您需要：

1. 从水龙头（Faucet）获取测试网 ETH
2. 使用测试网 ETH 获取 TKA 和 TKB 测试代币

### 第三步：开始使用

连接钱包后，您可以：
- 进入 **"交换"** 页面进行代币交易
- 进入 **"流动性"** 页面添加资产赚取收益
- 进入 **"池子"** 页面查看池子状态
- 进入 **"Bot 记录"** 页面查看自动复投记录

## 核心功能详解

### 🔄 代币交换

代币交换允许您将一种代币换成另一种代币。

#### 如何交换代币

1. 访问 **"交换"** 页面
2. 选择要交换的代币类型（TKA 或 TKB）
3. 输入要交换的数量
4. 系统会自动计算您将收到的代币数量
5. 查看价格和手续费信息
6. 点击 **"交换"** 按钮
7. 在钱包中确认交易

#### 关键信息说明

- **价格**：当前的交换比例（例如：1 TKA = 1.000 TKB）
- **滑点容差**：价格波动保护，默认 0.5%
- **手续费**：每笔交易收取 0.3%

#### 💡 小贴士

- 交换大额资产时可能会影响价格
- 建议先进行小额测试交易
- 注意留足够的 ETH 支付 Gas 费用

---

### 💧 提供流动性

通过提供流动性，您可以将资产存入池子，赚取交易手续费收益。

#### 如何添加流动性

1. 访问 **"流动性"** 页面
2. 点击 **"添加流动性"** 标签
3. 输入要添加的 TKA 数量
4. 系统会自动计算需要的 TKB 数量（保持比例）
5. 查看您将获得的 LP Token 数量和池子份额
6. 点击 **"添加流动性"** 按钮
7. 在钱包中确认交易

#### 关键信息说明

- **当前价格**：当前池子中两种代币的比例
- **池子份额**：您在整个池子中所占的比例
- **LP Token**：代表您流动性份额的凭证

#### 收益来源

当其他用户进行代币交换时：
- 每笔交易收取 0.3% 手续费
- 手续费按比例分配给所有流动性提供者
- 智能 Bot 会自动将手续费复投，增加您的收益

#### 如何移除流动性

1. 访问 **"流动性"** 页面
2. 点击 **"移除流动性"** 标签
3. 输入要移除的 LP Token 数量
4. 可以使用快捷按钮选择 25%、50%、75% 或 100%
5. 查看您将收到的 TKA 和 TKB 数量
6. 点击 **"移除流动性"** 按钮
7. 在钱包中确认交易

#### 💡 小贴士

- 添加流动性需要同时提供两种代币
- LP Token 是您提取资产的凭证，请妥善保管
- 移除流动性时，收到的代币数量可能与存入时不同（因为价格会波动）

### 📊 查看池子信息

在 **"池子"** 页面，您可以查看：

#### 池子概况

- **总锁定价值（TVL）**：池子中所有资产的总价值
- **24小时交易量**：过去24小时的交易总额
- **24小时手续费**：过去24小时产生的手续费收入

#### 储备量信息

- **Token A 储备**：池子中 TKA 的数量
- **Token B 储备**：池子中 TKB 的数量
- **当前价格**：实时的代币交换比例

#### 价格历史图表

- 查看不同时间段的价格走势
- 可选择查看 24小时、7天 或 30天 的数据
- 帮助您了解价格变化趋势

#### 最近交易记录

- 查看最近发生的所有交易
- 包括交易时间、类型、金额等信息

### 🤖 自动复投机制

这是 Mini-AMM 的特色功能！您无需任何操作，智能机器人会自动帮您增加收益。

#### 工作原理

**自动复投流程：**

1. 用户进行代币交换 → 产生 0.3% 手续费
2. 手续费累积在池子中
3. Bot 每 5 分钟检查一次累积的手续费
4. 自动将手续费重新投入池子
5. 增加所有 LP Token 的价值

**价格再平衡：**

1. Bot 每分钟监控价格偏差
2. 当价格偏离超过 5% 时自动触发
3. 执行小额交易调整池子比例
4. 使价格回归合理区间

#### 查看 Bot 操作记录

在 **"Bot 记录"** 页面，您可以：

- 查看 Bot 运行状态
- 查看历史复投记录
- 查看再平衡操作记录
- 筛选查看特定类型的操作

#### Bot 配置信息

- **复投间隔**：5 分钟
- **再平衡间隔**：1 分钟
- **再平衡阈值**：5% 价格偏差

#### 💡 关键优势

- ✅ **全自动运行**：无需您手动操作
- ✅ **持续增值**：收益不断复投
- ✅ **价格稳定**：自动调整保持平衡
- ✅ **24/7 运行**：全天候不间断工作

## 常见问题

### Q1: 我需要多少资金才能开始？

**答**：没有最低限制，但建议至少准备：
- 少量测试网 ETH 用于支付 Gas 费用
- 一些 TKA 和 TKB 代币用于交换或提供流动性

### Q2: 手续费是如何计算的？

**答**：
- **交易手续费**：每笔交换收取 0.3% 
- **Gas 费用**：由以太坊网络收取，金额取决于网络拥堵情况
- 手续费会自动分配给流动性提供者

### Q3: 提供流动性有什么风险？

**答**：主要风险是 **无常损失**：
- 当池子中两种代币的价格比例发生变化时
- 您取回的代币价值可能低于直接持有
- 但手续费收益可以弥补部分或全部损失

建议：
- 先用小额资金测试
- 了解价格波动风险
- 长期持有可以获得更多手续费收益

### Q4: LP Token 是什么？

**答**：
- LP Token 是流动性提供者代币
- 它代表您在池子中的份额
- 类似于存款凭证
- 移除流动性时需要 LP Token

### Q5: 为什么我的交易失败了？

**答**：常见原因：
- Gas 费用不足
- 滑点设置太低（价格变化超过允许范围）
- 余额不足
- 网络拥堵

解决方法：
- 确保钱包中有足够的 ETH 支付 Gas
- 增加滑点容差
- 检查代币余额
- 稍后重试

### Q6: 自动复投的收益如何体现？

**答**：
- 收益不会直接发到您的钱包
- 而是增加了您的 LP Token 价值
- 当您移除流动性时，会收到更多代币
- 可以在"池子"页面查看 LP Token 的当前价值

### Q7: 我可以随时取出资金吗？

**答**：
- 是的，您可以随时移除流动性
- 需要在"流动性"页面操作
- 使用 LP Token 换回原始代币
- 没有锁定期限制

### Q8: 如何查看我的收益？

**答**：查看收益的方法：
1. 在"流动性"页面查看您的头寸
2. 在"池子"页面查看累积手续费
3. 在"Bot 记录"页面查看复投记录
4. 计算您当前可取回的代币量与初始投入的差额

### Q9: 测试网代币有价值吗？

**答**：
- 测试网代币**没有实际价值**
- 仅用于学习和测试
- 可以免费从水龙头获取
- 不能用于真实交易
